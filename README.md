# Starstreet Forecast

Official repository for Starstreet Forecast app releases for GNU/Linux, Windows and Mac.

Source Code and Website at https://www.starstreet.space/app
---

## 📋 Requirements

### GNU/Linux
- You need Qt6 for the Qt6 version. It will run on Debian 12>
- The GTK2 version will run on Ubuntu 16.04> on AMD_64. On aarch64 it will run on Debian 11 or greater.
- You need to install libtls-dev to be able to use the HTTPS functions

### Windows
- Windows 8 or later
- You need to install Microsoft Visual C++ 2013 to be able to launch the app
- OpenSSL is needed if you want to use HTTPS

### macOS
- macOS 13 (Ventura) or later
